# SmartVanilla

SmartVanilla is a lightweight JavaScript template library that allows the building of vanilla JavaScript 
SPA (Single Page Applications) making it easier to build modern web applications without relying on heavy frameworks.

## Features

- Lightweight and fast
- Easy to learn and use
- Modular architecture
- Supports modern JavaScript features
- No dependencies on large frameworks
- Uses Bun as the package manager
- Uses Tailwind CSS for styling


## Installation

This template uses Bun as the package manager. You may need to install Bun first. 
You can find the installation instructions on the [Bun website](https://bun.sh/).

You can install SmartVanilla via npm:

```bash
bun install
```

## Usage

To start a development server, run:

```bash
bun run start
```
